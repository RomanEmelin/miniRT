# miniRT
A basic ray tracing project, which can display various primitive shapes like square, plane, cylinder, triangle and sphere.

## Installation:
>https://github.com/RomanEmelin/miniRT
The Makefile will compile the project properly.

## Usage:
>./minirt <scenename.rt>
